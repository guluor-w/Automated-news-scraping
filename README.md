# Automated-news-scraping

自动化抓取人工智能、制造业、工业化相关新闻，以进行供需平台的搭建。

### 当前数据来源

1、工业和信息化部官网https://www.miit.gov.cn/
2、中国政府网https://www.gov.cn/
3、腾讯网_工信微报账号发表的文章(经过对比，和公众号上面的内容重叠且少滞后性)

ps：在config.yaml的sources里可以增加来源

### 执行效果

在weekly.yml里面设置，每周周一 01:00定期触发从以上来源收集相关新闻。
经过比对去重后追增到data/policy_news.csv里，并按照发布时间降序排序（若发布时间缺失，则按照抓取时间排列），即新增的新闻会在csv文件的最上方；
另外每次运行会在added_count.txt里面记录新增的条数（覆写记录，只会显示最新一次运行的增加条数）。

### 参数配置（config.yaml）

在config.yaml里，可以增删关键词、修改时间窗口。

```
keywords:   #可以增加
  - 人工智能
  - AI
  - 大模型
  - 智能制造
  - 制造业
  - 新型工业化
  - 工业互联网
  - 数字化
  - 算力
  - 机器人

# 时间窗口（要求不要超过半个月）
window_days: 15  #如果每周连续运行一般以7天为窗口，这里先设置和hard_cap_days一致
hard_cap_days: 15 #最早时间（用于筛掉过早的新闻）
```
